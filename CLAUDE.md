# Design Context

Full strategic and visual specs live in [`PRODUCT.md`](PRODUCT.md) and [`DESIGN.md`](DESIGN.md) — read both before making design decisions on this project.

- **Register:** brand (marketing landing page — design IS the product) · **Platform:** web
- **What this is:** personal landing page for Priscila Cueva (digital strategy, automation, web design, UGC content), bilingual ES/EN, single file at `Priscila Cueva.dc.html`
- **Positioning:** solutions designed 100% for each business, never generic templates
- **North Star (visual):** "La Hora Dorada" (The Golden Hour) — near-black warm base (`#0b0a08`), aged gold accent (`#cda86a`) used sparingly, Playfair Display serif headlines + Manrope sans body
- **Primary CTA:** start a WhatsApp conversation. **Don't** ever introduce urgency tactics, countdowns, or exaggerated testimonials — see PRODUCT.md anti-references.
